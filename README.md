Netflix Data Analysis using Python and Pandas

 📌 Project Overview
This project analyzes the Netflix Titles dataset using Python and Pandas. It includes data cleaning, exploratory data analysis (EDA), and visualizations to understand trends in Netflix content.

 🎯 Objectives
- Analyze Movies and TV Shows
- Identify the top countries producing Netflix content
- Find the most popular genres
- Analyze content ratings
- Study release year trends

 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
- Google Colab

 📊 Visualizations
- Movies vs TV Shows
- Top 10 Countries
- Top 10 Genres
- Ratings Distribution
- Release Year Analysis

 📈 Key Insights
- Netflix has more Movies than TV Shows.
- The United States has the highest amount of content.
- Drama and International Movies are among the most common genres.
- Most Netflix content was released in recent years.

 🚀 How to Run
1. Open the notebook in Google Colab or Jupyter Notebook.
2. Install the required libraries.
3. Upload the Netflix dataset.
4. Run all cells.
